## Overview

This assignment covers two complementary testing approaches: **mutation testing** using PIT (Pitest) on the JFreeChart `Range` and `DataUtilities` classes to evaluate and improve existing JUnit test suites, and **GUI testing** using Selenium IDE to automate browser-based tests for a web application.

## Structure

- `JFreeChart_Lab4/` - JFreeChart project with source code and JUnit test suites (includes `RangeTest` and `DataUtilitiesTest`)
- `selenium-scripts/` - Selenium IDE test scripts for GUI testing (Walmart)
- `media/` - Screenshots used in the report and instructions
